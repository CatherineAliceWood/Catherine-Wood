Wonderful Wool

MySQL database

A prototype database for a yarn shop with 3 tables - yarn, purchases and loyalty card holders, linked with foreign keys. Includes a stored procedure to process purchases. The script has queries to insert data, retrieve data, delete data, use aggregate functions, joins and in-built functions. The data is normalised by being split into tables with no duplicate data.

Star Wars Film Selector

Python console app using an API

A Python console app which helps you choose which Star Wars film to watch based on which characters you want to see, using the SWAPI API. If you don't add Luke Skywalker, it'll ask you if you want to. After you've finished choosing characters it gives you a list of the films which they're all in together and writes this info to a file.
